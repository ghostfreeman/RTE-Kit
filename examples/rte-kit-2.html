Example 2: Static Markup

This example is the same as Example 1, but prevents the page-load "jump" by using pre-written widget markup. The markup is the same as what would be generated by javascript automatically. The iframe is left for javascript to create automatically since making it ahead of time doesn't affect the height of the widget.

Markup

<div class="ui-rtekit"> <div class="ui-rtekit-toolbar"></div> <div class="ui-widget-content"> <textarea id="example-2"></textarea> </div> </div>

Javascript

$('#example-2').rteKit({ toolbar: 'bold italic underline | insert-paragraph remove-format' });

Example 3: Advanced Controls

Some designMode commands require a parameter from the user. This example uses select elements to choose the font face and size. The javascript prompt() procedure is used to create links and images. A custom command is also implemented to toggle the editor between source and rich-text modes.

Markup

<div class="ui-rtekit"> <div class="ui-rtekit-toolbar"></div> <div class="ui-widget-content"> <textarea id="example-3"></textarea> </div> </div>

Javascript

The menu() and with_prompt() procedures are defined in ./js/rte-kit.js.

function example3() { $('#example-3').rteKit({ toolbar: ( 'font-name font-size' + ' | bold italic underline insert-paragraph' + ' | create-link insert-image' + ' | remove-formatting html' ), tools: { 'create-link': with_prompt('Enter Link:'), 'insert-image': with_prompt('Enter Image URL:'), 'font-size': menu('Size', [1, 2, 3, 4, 5, 6, 7]), 'font-name': menu('Font', ['Arial', 'Times', 'Verdana', 'Helvetica', 'Sans']), 'html': $.ui.rteKit.button(function(event, rte, cmd) { rte.toggleFrame(); }) } }); }
